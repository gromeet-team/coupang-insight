# TASK: 쿠팡 인사이트 웹 페이지 제작

## 목적
쿠팡 판매 성공 전략 및 노하우를 정리한 웹 페이지 제작.
승우님과 공유할 용도 — 가독성 좋게, 전문적으로, 모바일 대응.

## 파일
- **출력**: `index.html` (단일 파일 — 외부 JS 없이 순수 HTML+CSS)
- **폰트**: Google Fonts CDN (Noto Sans KR)
- **아이콘**: 이모지 사용 (외부 라이브러리 없음)

## 디자인 방향
- 쿠팡 브랜드 컬러: #C0392B (레드), 보조: #E8E8E8 (그레이)
- 다크 헤더 + 라이트 본문 스타일
- 섹션 카드 형태로 구성
- 반응형 (모바일/태블릿/데스크탑)
- 깔끔하고 전문적인 비즈니스 문서 느낌

## 페이지 구조

### 헤더
- 제목: "쿠팡 성공 전략 가이드 2025"
- 부제목: "쿠팡에서 살아남는 법 — 핵심 노하우 총정리"
- 작성일: 2026년 3월

### 목차 (사이드바 or 상단)
1. 알고리즘 & 상위노출
2. 광고 운영 (쿠팡애즈)
3. 리뷰 전략
4. 상품 최적화
5. 가구매 현황 & 리스크
6. 슬롯 광고
7. 분석 도구
8. 2025년 트렌드

### 섹션별 콘텐츠

#### 1. 🔍 알고리즘 & 상위노출 원리
- 노출점수 시스템 설명 (80점 이상 → 상위노출)
  - 출고일 (짧을수록 유리)
  - 상품점수 (필수정보 완성도)
  - 가격경쟁력
- 알고리즘 4대 요소: 카테고리 + 상품명 + 구매 옵션 + 검색어 태그
- 중요 수치: "순위 1단계 상승 → 매출 23% 증가"
- 로켓배송 vs 로켓그로스 vs 마켓플레이스 비교 테이블
- 2025년 변화: 비검색 영역 경쟁력 부상, 실 구매 행동 데이터 중시

#### 2. 📣 광고 운영 (쿠팡애즈)
- 자동 vs 수동 입찰 비교 테이블 (황금 비율 60:40)
- 골든타임: 오후 6~9시 클릭 1.7배
- ROAS 기준점: 650% 이상 → 알고리즘 가속 효과
- 광고 그룹 구조: 브랜드(20%) / 상품(30%) / 카테고리(50%)
- 키워드 유형: 헤드 / 미들 / 롱테일 전략

#### 3. ⭐ 리뷰 전략
- 합법적 리뷰 확보 방법
  1. 쿠팡 공식 체험단
  2. 동봉 안내 카드 활용
  3. 제품 품질 + CS 응대
- 탑100 리뷰어 활용 (권위 리뷰)
  - 쿠팡에서 활용 가능한 서비스: 주식회사 디디하이브 (이서우 대표, 010-9824-8848)
  - 단가: 일반 리뷰 3,000원 / 탑 리뷰 5,000원 (건당)
  - 주의: 모든 제품은 반드시 실제 배송 필요, 타사 제품 직접 언급 금지
- 물길리뷰 전략 (경쟁사 리뷰어 리다이렉션 개념)
- 부정 리뷰 대응법

#### 4. 🛒 상품 최적화
- 제목 최적화: 핵심 키워드 앞 20자 배치 공식
- 검색어 태그 전략 (롱테일 위주 20개)
- 아이템위너 제도와 자동가격조정
- 이미지 최적화 (화이트 배경, 1000px 이상)

#### 5. ⚠️ 가구매 현황 & 리스크
- 가구매란 무엇인가
- 2019~2020 vs 2024~2025 탐지 수준 변화
- 계정 정지 위험 경고 박스 (WARNING 스타일)
- 합법적 대안:
  1. ✅ 가구매 실 가격 구매 시 현재 효과 있음 (하루 1-20개, 100개+ 시 상위노출)
  2. 주의: 동일 배송지 반복 탐지 위험
  3. 합법 대안: 공식 체험단, 초기 광고 집중, 외부 트래픽

#### 6. 🎯 쿠팡 슬롯 광고
- 슬롯 광고 개념 설명
- 시그니쳐 슬롯: 해당 키워드만 상승, 비교적 안정
- 쿠팡슬롯 효과: 5-10개씩 물린 키워드 기준 1p 진입 (실제 운영 경험)
- 리스크: 쿠팡 정책 위반 소지, 장기 지속 어려움
- 2025년: 효과 감소 추세

#### 7. 📊 판매량 분석 도구
- 도구 비교 테이블:
  - 쿠파일럿 (coupilot.net) — 키워드 분석
  - 아이템스카우트 (itemscout.io) — 키워드 분석
  - mallscore.co.kr — 판매량 분석
  - 순위 트래킹 툴 (자체제작, 월 3만원) — 실행사/대행사들이 탐내는 툴
  - SellDataLab — 판매량 분석
  - 주디노 (크몽) — 경쟁사 데이터
  - SMART DATA 크롬 확장

#### 8. 📈 2025년 트렌드 & 전략
- 시장 현황: 영업이익 9,850억원, 활성 고객 2,470만명
- 로켓그로스 수수료 개편 (물류비 6단계 세분화)
- 2년 전 vs 2025년 전략 비교 테이블
- 성공 공식 다이어그램:
  "검색어 태그 최적화 → 저경쟁 키워드 발굴 → 골든타임 광고 → ROAS 관리 → 체험단 리뷰 → 데이터 최적화"
